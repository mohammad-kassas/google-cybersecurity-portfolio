# Security Audit — Botium Toys

## Task
Conducted an internal security audit for Botium Toys,
a fictional small U.S. toy company expanding online.

## What I Did
- Reviewed scope, goals, and risk assessment report
- Identified assets managed by the IT department
- Evaluated existing controls against NIST CSF
- Completed a controls and compliance checklist
- Assessed compliance with PCI DSS, GDPR, and SOC

## Key Findings
- No encryption on customer credit card data
- All employees had access to sensitive PII (no least privilege)
- No disaster recovery plan or data backups
- No intrusion detection system (IDS)
- Weak password policy with no central management

## Tools/Frameworks Used
- NIST Cybersecurity Framework (CSF)
- PCI DSS, GDPR, SOC compliance standards

## Recommendation
Immediate priorities: implement encryption, least privilege
access controls, IDS, and a disaster recovery plan.
